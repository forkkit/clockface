# Tabs with Links and Delete buttons

This is an example of `Tabs` in which some tabs are using the `linkElement` prop and others are using the `onDismiss` prop.

### Usage
```tsx
import {Tabs} from '@influxdata/clockface'
```
```tsx
<Tabs>
  <Tabs.Tab />
  <Tabs.Tab />
  <Tabs.Tab />
```
